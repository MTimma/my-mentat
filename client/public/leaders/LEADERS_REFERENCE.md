# | Dune: Imperium — Leader Abilities & Signet Ring Reference

Parsed text from leader cards. Images are in the same folder.

## Leader head icons

Cropped head/portrait icons (128×128) for UI use are in `icons/`. Generate them with:

```bash
cd client && npm run crop-leader-heads
```

Uses macOS `sips` (no extra deps, AVIF 2.0 compatible). Output: `leaders/icons/{name}-head.png` (128×128).
