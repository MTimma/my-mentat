## Bugs & regressions

### BUG-001 (RETRACTED) — “Agent placement completely broken”

**Verdict:** False alarm from session 1.  
Red prohibition circles on **non-matching** board spaces are correct. The Voice uses **CITY + SPICE_TRADE** icons (not Landsraad). Placement on **Carthag** succeeded in session 2.

**UX follow-up:** No tooltip on disabled hotspots explaining icon mismatch, cost, or occupancy.

### BUG-002 — The Voice blocks End Turn without clear guidance

**Severity:** Major (UX / rules flow)  
**Steps:** Play The Voice → place agent → board rewards auto-claim → **THE_VOICE** reward remains (choose space to block opponents).  
**Expected:** Prominent prompt to pick a blocked space; `canEndTurn` stays false until claimed (per `theVoice.test.ts`).  
**Actual:** End Turn appears available or turn feels “stuck”; easy to misread as turn-progression failure.  
**Screenshot:** `03-agent-placed-carthag-success.webp`, `04-player1-turn-complete.webp`

### BUG-003 — Optional / cost board-space effects may block End Turn

**Severity:** Major (possible functional)  
**Steps:** Agent on Fremen combat space (reported as Stillsuits; **Hardy Warriors** has mandatory 1 water cost in data). End Turn toggles water-cost UI; turn does not advance.  
**Expected:** Pay cost, decline optional arrow costs, or clear pending optional effect, then End Turn.  
**Actual:** Repeated End Turn clicks cycle UI without advancing to next player.  
**Needs:** Dev repro on Hardy Warriors / optional-cost card effects; add Skip/Decline for optional costs.

### BUG-004 — No auto-resume after page refresh

**Severity:** Minor (by design?)  
Refresh returns to home; no localStorage session. Manual export exists via turn-history debug / download save.  
**Screenshot:** `05-session-end-home-screen.webp`

### BUG-005 — `buildHistory` duplicate row on rejected END_TURN

**Severity:** Minor (automated)  
`save/__tests__/buildHistory.test.ts` failing — aligns with undo/history edge case in plan UI-34.
